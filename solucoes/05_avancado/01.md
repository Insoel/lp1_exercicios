## Problema

1 - Escreve um método `static` que troque o valor de duas variáveis de entrada
cujo tipo é definido em tempo de execução (ou seja, por quem invoca o método).

## Soluções

### Solução 1

```cs
static void ChangeValues(object var1, object var2)
{
	var temp = var1;
	var1 = var2;
	var2 = temp;
}
```
